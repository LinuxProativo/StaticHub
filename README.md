 
# StaticHub

This repository contains statically linked binaries compiled for portability and ease of distribution.

## About

- All binaries are built to be self-contained with no external shared library dependencies.
- Intended for use in minimal, embedded, or containerized environments.
- Each release includes verified static binaries for supported platforms.
